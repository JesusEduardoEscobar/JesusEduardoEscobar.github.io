HTML

Introducción

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Se utiliza para describir la estructura de las páginas web mediante el uso de etiquetas.

¿Qué es HTML?

HTML es el lenguaje de marcado de hipertexto que se utiliza para crear páginas web. Define la estructura y el contenido de las páginas web.

Explicación de las etiquetas head y body

La etiqueta head contiene metadatos y enlaces a recursos externos, como hojas de estilo y scripts. La etiqueta body contiene el contenido visible de la página web, como texto, imágenes y otros elementos.

Etiquetas Básicas

Explicación de las etiquetas section, article, aside, header, footer

  • section: Define una sección del documento.
  • article: Define un artículo o contenido independiente.
  • aside: Define contenido que está relacionado pero no es parte del contenido principal.
  • header: Define un encabezado para una sección o página.
  • footer: Define un pie de página para una sección o página.

Introducción

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje el cual nos permite agregar diseño a nuestra pagina

Propósito de CSS

Cómo aplicar CSS a HTML (3 opciones)

  • CSS en línea: Se aplica directamente a los elementos HTML mediante el atributo style.
  • CSS interno: Se coloca dentro de la etiqueta style en el head del documento HTML.
  • CSS externo: Se coloca en un archivo CSS separado y se enlaza al documento HTML mediante la etiqueta link.

¿Qué es un selector? ¿Qué es una regla CSS?

Un selector es una cadena que indica a qué elementos HTML se aplica una regla CSS. Una regla CSS consta de un selector y un bloque de declaraciones que definen los estilos a aplicar.

¿Qué es una clase? ¿Qué es un ID? ¿Cómo se relacionan con CSS?

Una clase es un atributo que se puede asignar a uno o más elementos HTML para aplicarles estilos específicos mediante CSS. Un ID es un atributo único que se asigna a un solo elemento HTML. Estos se relacionan ya que con ambos podemos aplicarle estilo a un elemento ne especifico de nuestra pagina.

Explicación del modelo de caja CSS y ejemplo

El modelo de caja CSS describe cómo se dispone el contenido en una página web. Cada elemento se representa como un cuadro rectangular que tiene márgenes, bordes, relleno y contenido.
